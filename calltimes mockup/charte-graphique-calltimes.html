Règles d'Usage

✓ À FAIRE
  • Utiliser Inter Black pour les appels à l'action et titres forts
  • Réserver Baskerville Italic pour les moments élégants et narratifs
  • Maintenir le contraste fort entre les deux fonts
  • Uppercase uniquement pour Inter (jamais Baskerville)
  • Espacements généreux autour des titres Baskerville
  • Tracking négatif sur Inter Black pour les gros titres
✗ À ÉVITER
  • Utiliser Baskerville pour des boutons ou labels UI
  • Mettre Baskerville en uppercase (perd son élégance)
  • Utiliser Inter Regular pour des titres (pas assez d'impact)
  • Mélanger les deux fonts dans une même phrase
  • Utiliser Baskerville en petite taille (< 14px)
  • Trop d'italiques sur une même page (dilue l'effet)

Implémentation Technique

Google Fonts Import
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Libre+Baskerville:ital,wght@0,400;0,700;1,400&display=swap');

/* Logo */
.logo-call {
  font-family: 'Inter', sans-serif;
  font-weight: 900;
  text-transform: uppercase;
  letter-spacing: -0.02em;
}

.logo-times {
  font-family: 'Libre Baskerville', serif;
  font-style: italic;
  font-weight: 400;
}

/* Hero Title */
.hero-title {
  font-family: 'Inter', sans-serif;
  font-weight: 900;
  font-size: 4rem;
  text-transform: uppercase;
  letter-spacing: -0.02em;
  line-height: 0.9;
}

/* Section Header */
.section-header {
  font-family: 'Libre Baskerville', serif;
  font-style: italic;
  font-size: 1.75rem;
}
